XMAS_Peanuts

Content

  • BackupPeople - has the previous year solution in case of lost.
  • data - contains the data needed. You must fill in mail the data on the following format :
    >NAME ADDRESS
    Pay attention there is a tabular between both, they must have no blank line at the end.
  • People - contains the pdf file of who needs to offer who.
  • ASCII - contains the ascii art you want to add at your message.
  • clean - is a small script that cleans the folder People.
  • ConvertToPdf - is script converting into ps (postscript) and then into pdf.
  • Programme Noel.py - is the script dispatching the people.
  • run - is the run script.
  • sendmail - is the small script that sends the mail. One needs to change here the content of the mail.
  • 10/ template2017 - TODO

How to use it

  1. Enter the name and the address in data/mail
  2. Change the ASCII art in ASCII
  3. Change the mail in SendMail
